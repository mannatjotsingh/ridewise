# RideWise - Ride Sharing System

## Project Overview

RideWise is a console-based Ride Sharing System built using Core Java.

The application allows:
- Rider registration
- Driver registration
- Viewing available drivers
- Ride booking
- Driver assignment using different matching strategies
- Fare calculation using pricing strategies
- Ride completion and ride tracking

The project demonstrates:
- OOP concepts
- SOLID principles
- Strategy Design Pattern
- Composition over inheritance
- Layered architecture
- Java collections
- Exception handling
